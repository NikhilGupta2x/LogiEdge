#LogiEdge Billing Dashboard

## Live DEMO
-Frontend: https://logiedge-app.vercel.app
-Backend: https://logiedge.onrender.com

## How to run locally
### Database
Run `database.sql` in pgAdmin to create all tables

### Backend
cd "Backend Server" 
npm install 
node server.js

### Frontend
cd "Frontend App"
npm install
npm run dev
